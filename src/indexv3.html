I wanted to create something that others could use to contribute to the webdev community. I decided to develop a single page with 14 different hover effects aimed towards image galleries. Rather than create it just as a repo where you'd import the code for all 14 effects I used modals to show the code for the desired effect that can be copied and pasted for a more optimized solution.
